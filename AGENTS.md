# Bomberman Clone

17x17 Bomberman built in Jai (game.jai) using the Simp module for 2D rendering. We use Jai now — `game.js` is kept only as a reference for the original prototype. Build with `jai game.jai`, run `game.exe`. Player moves with WASD/Arrows, places bombs with Space/E, punches with Q. Player spawns top-left.
